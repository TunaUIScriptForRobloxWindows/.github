## 🛠️ Usage
```lua
local TunaUI = require(game:GetService("ReplicatedStorage").TunaUI)

-- Create a new button
local myButton = TunaUI.Button.new("Click Me!")
myButton:SetPosition(UDim2.new(0.5, 0, 0.5, 0))
```
